       - activities table: date (NOT NULL), activity_type, summary fields, raw_data JSONB, created_at

       - upsert_daily_metrics(client, record: DailyMetrics) -> None: calls .table("daily_metrics").upsert(record.model_dump(mode="json"), on_conflict="date").execute()

         f. Log warning if missing critical fields
         g. Get Supabase client
         h. Upsert daily metrics and activities
         i. Return IngestionResult

       - Also add __main__.py to src/biointelligence/ so `python -m biointelligence` works
